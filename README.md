# flutter_rspGame
## 가위바위보 게임

- 진행 규칙
  - 가위 바위 보 중 버튼을 선택하면 컴퓨터에서 랜덤으로 생성된 가위 바위 보와 비교하여 승패를 판별하며 게임 진행
  - 5판을 진행하면 게임 결과 창이 뜬다
  - 초기화 버튼을 누르면 모든 것이 초기화 됨
